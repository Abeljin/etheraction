    简介

    Solidity

    Solidity 是面向智能合约的高级编程语言，它的设计受 C++、Python和JavaScript影响，旨在针对以太坊虚拟机(EVM)。

    Solidity 是静态类型语言，支持继承，库和复杂的自定义类型等功能。

    正如你见，它可用来创建投票，众筹，盲目拍卖，多签名钱包等等合同。

    Note

    现在，学习 Solidity 的最佳途径是用在线IDE Remix(可能需要加载一段时间完成准备，请耐心等待)。

    可用的 Solidity 集成

    • Remix 基于浏览器的IDE，含编译器和运行环境，无需服务端组件。

    • IntelliJ IDEA 插件 集成到Intillij IDEA 的Solidity插件（支持所有的 JetBrains IDE）。

    • Visual Studio 扩展 含 Solidity 编译器的 微软 Visual Studio 插件。

    • SublimeText包- Solidity语法高亮 SublimeText编辑器中支持对 Solidty 语法高亮。

    • Etheratom Atom 编辑器的 Solidity 插件，支持语法高亮，编译和一个运行时环境（后端节点与 VM 兼容）。

    • Atom Solidity Linter Atom 编辑下 Solidity 插件，可对 Solidity 代码检查分析。

    • Atom Solium Linter

    Atom 下，基于 Solium 的 Solidty 配置检查器。

    • Solium 一个严格遵循Solidity 代码风格指南规范的命令行 Solidity 代码检查分析器。

    • Solhint Solidity 代码检查分析器，对智能合约提供安全、风格指南和最佳实践检查。

    • Visual Studio Code 扩展 微软 Visual Studio Code 下 Solidity 插件，含语法高亮和 编译器。

    • Emacs Solidity Emacs 编辑器下 Solidity 插件，含语法高亮和编译错误提示。

    • Vim Solidity Vim 编译器下 Solidity 插件，支持编译检查。

    Solidity 工具集

    • Dapp Solidity 的构建工具、包管理器和部署助手。

    • [Solidity REPL])https://github.com/raineorshine/solidity-repl) 可立即在命令行尝试 Solidity。

    • solgraph 可视化 Solidity 控制流程并高亮突出显示潜在的安全漏洞。

    • evmdis EVM 反编译，对字节码进行静态分析，为字节码提供比原始 EVM 操作更高的抽象级别。

    • Doxity Solidity 文档生成器。

    • solidity-parser JavaScript版 Solidity 解析器。

    • Solidity Grammar for ANTLR 4 ANTLR4 解析器生成器的 Solidity 语法。
